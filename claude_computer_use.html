Computer Use

화면을 직접 보고 마우스·키보드를 조작하는 AI. 브라우저, 데스크탑 앱 등 어떤 소프트웨어든 사람처럼 제어해 반복 업무를 완전 자동화합니다.

⚠ Research Preview — Computer Use는 현재 Research Preview 단계로, Claude Pro 또는 Max 플랜이 필요합니다. macOS 및 Windows에서 지원됩니다.

사용 방법

1

요구 사항 확인

Computer Use를 사용하려면 Claude Pro 또는 Max 플랜이 필요합니다. Claude Code 또는 Claude Cowork 앱을 통해 접근할 수 있으며, macOS와 Windows를 지원합니다.

2

Claude Code에서 활성화

Claude Code 터미널에서 Computer Use 기능을 활성화합니다. 설정 메뉴에서 Computer Use 토글을 켜면 Claude가 현재 화면을 스크린샷으로 캡처하고 마우스·키보드를 제어할 수 있게 됩니다.

3

API로 직접 사용 (개발자)

Anthropic API를 통해 Computer Use 도구를 직접 호출할 수 있습니다. computer_20251022 도구를 사용하며, 스크린샷 캡처 → Claude 분석 → 액션 실행의 루프로 동작합니다.

# beta 헤더 필요
anthropic-beta: computer-use-2024-10-22

# 사용 가능한 도구
- computer: 화면 캡처, 마우스/키보드 제어
- bash: 터미널 명령 실행
- text_editor: 파일 편집
4

자연어로 작업 지시

사람에게 지시하듯 자연어로 작업을 요청합니다. Claude가 화면을 보고 스스로 클릭, 타이핑, 스크롤, 파일 조작을 수행합니다.

# 예시 지시
"보고서 초안을 PDF로 변환하고
오늘 오후 2시 회의 초대에 첨부해줘"

"바탕화면의 사진 150장 크기를
1080px로 통일하고 로고를 삽입해줘"
5

주의사항 및 보안

Claude가 화면 전체에 접근하므로 민감한 정보가 담긴 창은 미리 닫아두는 것이 좋습니다. 현재 단계에서는 오류가 발생할 수 있으며, 중요한 작업은 작업 후 결과를 반드시 검토하세요.
