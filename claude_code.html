Developer Tool

Claude Code

터미널에서 실행하는 AI 코딩 에이전트. 코드 읽기·수정·실행부터 Git 연동까지 개발 워크플로우 전반을 자율적으로 처리합니다.

사용 방법

1

설치

Node.js 18 이상이 설치된 환경에서 아래 명령어로 Claude Code를 전역 설치합니다.

npm install -g @anthropic-ai/claude-code
2

실행 및 인증

프로젝트 루트 디렉토리에서 claude를 실행하면 Anthropic 계정으로 로그인하라는 안내가 표시됩니다. 최초 1회 인증 후 자동으로 저장됩니다.

cd my-project
claude
3

CLAUDE.md 설정

프로젝트 루트에 CLAUDE.md 파일을 만들어 Claude가 항상 참조할 프로젝트 규칙, 코딩 컨벤션, 자주 쓰는 명령어 등을 정의합니다. /init 명령어로 자동 생성도 가능합니다.

/init
4

Plan Mode 활용

복잡한 작업 전에 Shift + Tab을 눌러 Plan Mode를 활성화합니다. Claude가 코드를 변경하기 전에 실행 계획을 먼저 보여주므로, 의도치 않은 수정을 방지할 수 있습니다.

5

주요 슬래시 커맨드

대화 중 슬래시 커맨드로 기능을 제어합니다.

/help          — 도움말
/clear         — 대화 초기화
/compact       — 컨텍스트 압축
/cost          — 현재 세션 비용 확인
/review        — 코드 리뷰 요청
6

Git 연동

Claude Code는 Git을 자동으로 인식합니다. "변경사항 커밋해줘", "새 브랜치 만들고 PR 올려줘" 같은 자연어 명령으로 Git 작업을 처리할 수 있습니다.
