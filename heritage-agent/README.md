# On-Go (to Heritage)

**Heritage AI Tour Guide Agent** - 전세계 어디서든 사용할 수 있는 AI 기반 맞춤형 여행 가이드

사용자가 건물/유적지 사진을 업로드하면 AI가 장소를 인식하고, 사용자 개인 프로필(나이, 성별, MBTI)에 맞춰 설명·음성 해설·주변 추천 정보를 제공하는 Streamlit 웹앱.

---

## 핵심 기능

### 1. 넷플릭스 스타일 프로필 시스템
- 여러 개의 사용자 프로필 생성/선택/수정/삭제
- 각 프로필에 이름, 나이, 성별, MBTI, 전문가 모드 저장
- 프로필별로 방문 기록과 추천이 독립적으로 관리됨

### 2. AI 장소 인식 (Gemini 2.5 Flash)
- 사진 업로드 → EXIF GPS 자동 추출
- Gemini Vision으로 전세계 건물/랜드마크 자동 식별
- 결과: 이름, 위치(도시/국가), 좌표, 카테고리, 확신도, 판단 근거
- 대상: 대학, 종교건축, 궁전, 유적지, 현대건축, 탑, 다리, 기념물 등 전세계 모든 장소

### 3. 페르소나 맞춤 AI 설명 + TTS
- 나이대별 어투 (어린이/청소년/성인/전문가 모드)
- 성별 관점 힌트 (건축 기술 vs. 예술적 아름다움)
- MBTI 16유형 기반 스타일링 (E/I, S/N, T/F, J/P 4개 축)
- edge-tts 기반 페르소나별 한국어 TTS 음성 (여성/남성/전문가)
- 3단계 TTS 폴백: edge-tts 페르소나 → edge-tts 기본 → gTTS

### 4. 주변 추천 (3종)
사진 설명을 받은 후 같은 장소 주변의 3가지 추천을 제공:

#### 4-1. 주변 유적지 (가이드 탭 내)
- 반경 30km 이내 유적지/랜드마크 **5개** 추천
- 페르소나 기반 (어린이=체험형, 청소년=SNS감성, MBTI별 취향)
- "자세히 보기" 클릭 시 사진 + 블로그 리뷰 + 유튜브 리뷰 링크

#### 4-2. 추천 맛집 (🍽️ 탭)
- 리뷰/방문자 수 기반 인기 맛집 **3개** 추천
- 카테고리 아이콘 (한식/양식/카페/디저트 등)
- 가격대 (💵/💵💵/💵💵💵), 인기도 (⭐⭐⭐)
- 페르소나 맞춤 (연령대+MBTI 기반 취향 반영)

#### 4-3. 주변 여행지/액티비티 (🎡 탭)
- 공원/테마파크/전망대/쇼핑/야경/테마거리/투어/체험/스파/공연/동물원/수족관/놀이공원 등 **3개** 추천
- ⚠️ 유적지/궁전/종교건축/기념물/박물관은 제외 (이미 가이드 탭에서 별도 추천)
- 소요시간 및 난이도 표시 (쉬움/보통/활동적)
- 어린이=놀이공원/동물원, 60대=공원/전망대, MBTI 기반 스타일 매칭

**각 추천마다 공통 미디어 제공 (네이버 기반):**
- 📷 네이버 이미지 검색 링크
- 📝 네이버 블로그 리뷰 검색 링크
- 🗺️ 네이버 지도 위치 링크 (장소 이름 + 지역명 기반 정확한 위치 제공)

### 5. 🗺️ 지도 앨범 (프로필별)
- Folium 기반 전세계 지도에 방문지 마커 표시
- 카테고리별 마커 색상 (궁전=빨강, 종교건축=보라, 현대건축=녹색 등)
- 마커 클릭 시 사진 팝업 + 설명 프리뷰
- **현재 로그인한 프로필의 기록만 표시** (다른 사용자 기록 숨김)
- 방문 기록 목록에서 과거 설명 다시 보기 가능

### 6. 클라우드 데이터 영속성 (GitHub Gist)
- Streamlit Community Cloud 재배포/리붓에도 데이터 유지
- `GITHUB_TOKEN`과 `GIST_ID`를 secrets에 설정하면 자동 활성화
- 저장 대상: 프로필, 방문 기록, 사진(base64)
- 폴백: Gist 미설정 시 로컬 디스크 저장
- 수동 백업/복원: JSON 내보내기/가져오기 버튼 제공

## 화면 구성

### A. 프로필 선택 화면
- 저장된 프로필 카드 그리드 (최대 4열)
- 각 카드: 아이콘, 이름, 나이, 성별, MBTI, 전문가 배지
- 버튼 3개: 선택 / 수정 / 삭제
- 하단: 프로필 수정 폼 + 새 프로필 만들기 폼

### B. 메인 앱 (프로필 선택 후)
상단: 현재 프로필 정보 + 🔄 전환 버튼

**4개 탭 구조:**

```
📸 가이드  |  🍽️ 추천 맛집  |  🎡 액티비티  |  🗺️ 지도 앨범
```

#### 📸 가이드 탭
1. 사진 업로드
2. AI 장소 인식 결과 표시 (이름, 위치, 카테고리, 확신도, 근거, 좌표)
3. [🔍 설명 받기] 버튼
4. 페르소나 맞춤 설명 + TTS 음성 플레이어
5. 방문 기록 자동 저장
6. 🧭 주변 추천 유적지 5개 카드 목록
7. 카드 선택 시 사진 + 블로그 + 유튜브 링크

#### 🍽️ 추천 맛집 탭
- 가이드에서 방문한 장소 주변 인기 맛집 3곳
- 각 카드: 음식 카테고리, 가격, 인기도, 설명, 추천이유, 사진, 리뷰 링크

#### 🎡 액티비티 탭
- 주변 여행지/액티비티 3곳 (유적지·박물관 제외)
- 각 카드: 카테고리, 소요시간, 난이도, 설명, 추천이유
- 📷 사진 검색 / 📝 블로그 리뷰 / 🗺️ 네이버 지도 버튼

#### 🗺️ 지도 앨범 탭
- 현재 프로필의 방문지 Folium 지도
- 방문 기록 목록 (날짜 역순)
- 클라우드 저장 상태 표시
- 데이터 내보내기/가져오기

## 실행

### 로컬 실행
```bash
cd heritage-agent
pip install -r requirements.txt
streamlit run app.py
```

`.env` 파일에 Gemini API 키 설정:
```
GEMINI_API_KEY=your_key_here
```

### Streamlit Community Cloud 배포
1. GitHub 저장소 연결 후 `heritage-agent/app.py`를 메인 파일로 지정
2. **Secrets** 설정:

   ```toml
   GEMINI_API_KEY = "AIza..."

   # (선택) 클라우드 데이터 영속성
   GITHUB_TOKEN = "ghp_..."
   GIST_ID = "abc123def456"
   ```

### Gist 기반 영속성 설정 방법
1. GitHub → Gists → New gist → 빈 파일 생성 후 "Create secret gist"
2. URL에서 Gist ID 복사 (`https://gist.github.com/username/<GIST_ID>`)
3. GitHub → Settings → Developer settings → Personal access tokens → Generate new token
4. 스코프: `gist`만 체크
5. 생성된 토큰과 Gist ID를 Streamlit Secrets에 등록
6. 앱 재시작 → 지도 앨범 탭에서 ✅ 표시 확인

## 페르소나 맞춤 로직 요약

### 나이대별 설명 톤
- 12세 이하: 쉬운 말, 3~4문장, 레고/게임/만화 비유
- 13~18세: 교과서 연결, 4~6문장, 드라마 같은 인물 이야기
- 성인: 교양 있는 톤, 5~7문장, 역사·양식·인물
- 전문가 모드: 학술 용어, 7~10문장, 학계 해석

### 성별 관점 힌트 (전문가 모드 외)
- 남성: 건축 구조, 공법, 설계 철학
- 여성: 예술적 아름다움, 사람들의 이야기

### MBTI 기반 스타일 (16유형)
- E/I: 활기찬 대화 / 차분한 깊이
- N/S: 상상력 비유 / 구체적 사실
- F/T: 감정적 공감 / 논리적 분석
- P/J: 자유로운 톤 / 체계적 정리

### 추천 페르소나 맞춤
- **유적지**: 어린이=체험형, 청소년=SNS감성, MBTI E/I/N/S/F/T 취향 매칭
- **맛집**: 어린이=키즈메뉴, 청소년=가성비, 50+=정갈, MBTI별 분위기 선호
- **여행지/액티비티** (유적지 제외): 어린이=놀이공원/동물원/수족관, 60+=공원/전망대, MBTI별 활동 성향

최초 기획 대비 주요 변경사항:

| 항목 | 최초 기획 | 현재 |
|---|---|---|
| 대상 | 홍릉/의릉 고정 유적지 | 전세계 모든 건물/랜드마크 |
| 페르소나 | 3종 (어린이/일반/역사 마니아) | 커스텀 프로필 (나이/성별/MBTI/전문가) |
| 추천 기능 | 없음 | 유적지 5개 + 맛집 3개 + 여행지/액티비티 3개 (유적지 제외) |
| 미디어 | 없음 | 네이버 이미지 검색 + 블로그 리뷰 + 네이버 지도 위치 |
| 데이터 저장 | 로컬 JSON만 | GitHub Gist 클라우드 + 로컬 이중 저장 |
| TTS | 단일 음성 | 페르소나별 + 3단계 폴백 |
| 지도 앨범 | 전체 공유 | 프로필별 분리 |
| UI | 단일 화면 | 프로필 선택 화면 + 4개 탭 메인 |
