Artifacts

대화 중 생성되는 구조화된 결과물. 코드·문서·UI를 별도 패널에서 즉시 미리보기하고 편집하며 공유까지 가능합니다.

사용 방법

1

Artifacts 활성화

claude.ai 우측 상단 설정 → Feature Preview에서 Artifacts를 활성화합니다. 활성화 후에는 Claude가 적합한 결과물을 생성할 때 자동으로 우측 패널에 렌더링합니다.

2

Artifact 생성 조건

Claude는 다음 조건을 충족하면 자동으로 Artifact를 생성합니다: 내용이 15줄 이상이거나, 반복 사용·편집 가능성이 높은 결과물이거나, 대화 외부에서도 독립적으로 사용 가능한 콘텐츠일 때입니다.

3

Artifact 유형

Claude가 생성할 수 있는 Artifact 유형은 다음과 같습니다.

💻
Code
모든 언어의 코드 스니펫, 함수, 스크립트
🌐
HTML
인터랙티브 웹페이지, 랜딩 페이지, 대시보드
⚛️
React
UI 컴포넌트, 인터랙티브 앱, 시뮬레이터
📊
Diagram
Mermaid 기반 플로우차트, 시퀀스 다이어그램
🖼️
SVG
벡터 그래픽, 아이콘, 기술 일러스트
📄
Markdown
구조화된 문서, 보고서, README
4

편집 및 반복 개선

Artifact 패널에서 직접 코드를 수정하거나, 대화창에서 "버튼 색을 파란색으로 바꿔줘", "모바일 반응형으로 만들어줘" 같이 지시하면 Claude가 즉시 Artifact를 업데이트합니다.

5

공유 및 재사용

완성된 Artifact는 공유 링크로 배포하거나, 다른 프로젝트에서 재사용할 수 있습니다. claude.ai/catalog/artifacts에서 커뮤니티 Artifact를 탐색하고 Remix할 수도 있습니다.
