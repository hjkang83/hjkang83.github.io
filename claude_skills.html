Claude Skills

슬래시 커맨드(/)로 호출하는 재사용 가능한 전문 에이전트. 반복 작업을 워크플로우로 패키징해 언제든 불러쓸 수 있습니다.

사용 방법

1

Skills 폴더 구조 이해

각 Skill은 .claude/skills/[skill-name]/ 폴더에 저장됩니다. 핵심 파일은 skill.md로, Claude가 해당 Skill을 실행할 때 참조하는 지시사항 파일입니다.

.claude/
  skills/
    my-skill/
      skill.md        ← 핵심 지시사항
      README.md       ← (선택) 설명
      templates/      ← (선택) 템플릿 파일
2

skill.md 작성

skill.md에는 이 Skill이 무엇을 하는지, 어떤 입력을 받는지, 어떤 결과를 출력하는지 자연어로 작성합니다. Claude가 이 내용을 읽고 실행합니다.

# Daily Report Skill

## 역할
매일 아침 팀의 작업 현황을 요약 보고서로 만든다.

## 입력
- $ARGUMENTS: 날짜 또는 기간

## 출력
- 어제 완료한 작업 목록
- 오늘 진행할 작업 계획
- 블로커 및 리스크 요약
3

Skill 호출

Claude Code 대화창에서 /[skill-name]을 입력하면 해당 Skill이 실행됩니다. 추가 인자를 전달할 수도 있습니다.

/daily-report 2026-04-04
/code-review src/main.py
/translate ko 이 문장을 번역해줘
4

공개 Skills 활용

Anthropic 공식 Skills 라이브러리 및 커뮤니티에서 제작한 Skills를 그대로 가져와 사용할 수 있습니다. claude.ai/skills에서 탐색하거나 GitHub 저장소를 클론해 .claude/skills/에 복사합니다.

5

팀 공유

프로젝트 Git 저장소에 .claude/skills/ 폴더를 커밋하면 팀 전체가 동일한 Skills를 공유합니다. 팀 내 표준 워크플로우를 Skills로 관리하는 것이 권장됩니다.
