Claude Features

What Claude
Can Do

Anthropic이 만든 AI 어시스턴트 Claude의 주요 기능들을 소개합니다.
단순한 대화를 넘어, 코드 작성부터 컴퓨터 제어까지 다양한 영역에서 활용할 수 있습니다.

01
Developer Tool

Claude Code

터미널에서 실행하는 AI 코딩 에이전트입니다. 코드를 읽고, 수정하고, 실행하며 복잡한 개발 작업을 자율적으로 처리합니다. 파일 시스템 접근, Git 연동, 테스트 실행까지 개발 워크플로우 전반을 지원합니다.

터미널 기반 CLI 에이전트 코드 읽기 / 수정 / 실행 Git 연동 및 자동 커밋 파일 시스템 탐색 멀티 파일 리팩토링
02
Automation

Claude Skills

슬래시 커맨드(/)로 호출하는 재사용 가능한 전문 에이전트입니다. 반복 작업을 자동화하거나 특정 도메인에 특화된 워크플로우를 정의해 언제든지 불러쓸 수 있습니다. 팀 단위로 공유하거나 프로젝트에 맞게 커스터마이징이 가능합니다.

/ 커맨드로 즉시 호출 재사용 가능한 워크플로우 커스텀 에이전트 정의 팀 공유 및 협업
03
Workspace

Claude Projects

지식베이스와 대화 맥락을 지속적으로 유지하는 워크스페이스입니다. 문서, 코드, 데이터를 업로드해 Claude가 항상 참조할 수 있게 하고, 여러 대화에 걸쳐 일관된 컨텍스트를 유지하며 작업할 수 있습니다.

영구적인 지식베이스 문서 / 코드 업로드 대화 간 컨텍스트 유지 커스텀 지시사항 설정 팀 프로젝트 공유
04
Multi-Agent

Dispatch

복잡한 작업을 여러 에이전트에게 자동으로 분배하는 오케스트레이션 기능입니다. 하나의 큰 목표를 병렬 서브태스크로 나누고, 각 에이전트의 결과를 취합해 더 빠르고 정확하게 목표를 달성합니다.

멀티 에이전트 오케스트레이션 병렬 태스크 처리 자동 작업 분배 결과 취합 및 통합
05
Autonomous

Computer Use

화면을 직접 보고 마우스와 키보드를 조작하는 AI 기능입니다. 브라우저, 데스크탑 앱, 웹 서비스 등 어떤 소프트웨어든 사람처럼 직접 제어하여 반복적인 업무를 완전 자동화할 수 있습니다.

화면 인식 및 분석 마우스 / 키보드 제어 브라우저 자동화 데스크탑 앱 조작 반복 업무 완전 자동화
06
Output

Artifacts

대화 중 생성되는 구조화된 결과물입니다. 코드, 문서, 차트, UI 컴포넌트 등을 독립적인 패널에서 즉시 미리보기하고 편집할 수 있습니다. 생성된 Artifact는 저장하거나 공유해 다른 프로젝트에서도 재활용할 수 있습니다.

코드 / 문서 즉시 미리보기 실시간 편집 UI 컴포넌트 생성 차트 및 시각화 저장 및 공유
